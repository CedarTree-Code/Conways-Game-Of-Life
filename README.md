# Conway's Game of Life

Using CUDA-OpenGL Interop to run Conway's Game of Life (B3/S23)

Cellular Automata are generally referred to as "Embarassingly Parallel" to code, A great fit for a first dive into CUDA kernels, blocks and threads

Using GLFW+glad for the OpenGL parts of the code

Changing Environment from WSL2 to Native Windows was an experience!
